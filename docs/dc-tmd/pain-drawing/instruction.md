# SCHMERZZEICHNUNG

## instruction

Bitte schraffieren Sie in den zutreffenden Zeichnungen ALLE Gebiete, in denen Sie unterschiedliche Schmerzen haben.

Wenn es einen genauen Punkt gibt, wo der Schmerz lokalisiert ist, kennzeichnen Sie ihn mit einem ausgefüllten Punkt (●). Wenn Ihr Schmerz von einem zu einem anderen Ort wandert, nutzen Sie Pfeile um den Weg aufzuzeigen.
