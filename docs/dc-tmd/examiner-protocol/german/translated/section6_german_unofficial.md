# 6 Untersuchungsbezogene Schmerzbefragung

## 6.1 Überblick

Die untersuchungsbezogene Schmerzbefragung ist ein strukturiertes Format zur wiederholten Erhebung und Klärung des Schmerzstatus bei positiven Untersuchungsbefunden. Die Absicht ist, dass der Untersucher in neutraler Weise nachfragt und der Patient die Beschreibung etwaiger erlebter Schmerzen liefert.

| **6.2.2** | **Für palpationsinduzierten Schmerz** | |
| | **Hatten Sie Schmerzen [in dem Bereich, in dem ich Druck ausgeübt habe]?** | |
| | FALLS JA zu Schmerz: | |
| | Weiter zu ABFRAGE BEKANNTER SCHMERZ. | |
| **6.2.3** | **Für knackbezogenen Schmerz** | |
| | **War dieses Knacken schmerzhaft?** | _Stellen Sie sicher, dass der Patient zwischen Schmerzen gleichzeitig mit dem Knacken und Schmerzen im Zusammenhang mit der Bewegung (d.h. Öffnen, Schließen, exkursive Bewegungen) unterscheidet._ |
| | FALLS JA zu Schmerz: | |
| | Weiter zu ABFRAGE BEKANNTER SCHMERZ. | |
| **6.2.4** | **Abfrage bekannter Schmerz** | |
| | **Ist dieser Schmerz Ihnen bekannt von Schmerzen, die Sie in diesem Bereich in den letzten 30 Tagen erfahren haben?** | _ANMERKUNG 1: Bezüglich „bekannt" kann der Untersucher etwa wie folgt erläutern: „Ist dieser Schmerz Ihnen bekannt, das heißt, ähnlich oder genauso wie der Schmerz, den Sie in diesem Bereich in den letzten 30 Tagen erfahren haben?" Die Erfassung des Konstrukts der Replikation ist nicht immer einfach, wie in Abschnitt 2.9 beschrieben._ |
| | | _ANMERKUNG 2: Wenn die durch die Untersuchung ausgelöste Schmerzintensität von der üblichen Schmerzintensität abweicht, fragen Sie: „Unabhängig von der Intensität, ist dieser Schmerz wie Ihr üblicher Schmerz?"_ |
| | | _ANMERKUNG 3: Während sich die Abfrage nach bekanntem Schmerz auf die letzten 30 Tage bezieht, geben manche Patienten möglicherweise freiwillig Informationen über mögliche Schmerzerfahrungen aus dem Zeitraum vor den letzten 30 Tagen an und berichten, dass ein bestimmter durch die Untersuchung ausgelöster Schmerz diesem Schmerz „bekannt" ist. Zum Beispiel könnte bei einer Person mit wiederkehrenden Schmerzen, die in den 30 Tagen vor einer Konsultation schmerzfrei war, die Palpation einen Schmerz auslösen, an den sich der Patient aus einer Schmerzepisode vor mehr als 30 Tagen erinnert. Der Standard-Zeitrahmen für diagnostisch relevante Schmerzen gemäß den DC/TMD sind die letzten 30 Tage. Unterschiedliche klinische oder Forschungssituationen können jedoch einen anderen Zeitraum erfordern. Folglich muss der Zeitrahmen für „bekannten Schmerz" möglicherweise angepasst werden. Notieren Sie jeden alternativen Zeitraum in Abschnitt 11 des Untersuchungsformulars._ |
| | FALLS JA ZU BEKANNTEM SCHMERZ: | |
| | **Wovon ist dieser Schmerz Ihnen bekannt?** | _ANMERKUNG 1: Bei den meisten klinischen Patienten ist die typische Antwort „der Schmerz, wegen dem ich hier bin". Sobald dieser Bezugsrahmen in den ersten Abfragen zu positivem Schmerz aus der Untersuchung etabliert ist, kann diese Frage üblicherweise weggelassen werden. Eine Ausnahme tritt auf, wenn zusätzliche Informationen aus der Anamnese darauf hindeuten, dass die der Schmerzbeschwerde zugrunde liegende Störung mit anderen möglichen Diagnosen überlappt; in diesem Fall kann die Nachfrage bezüglich der Schmerzreplikation umfangreicher sein._ |
| | | _ANMERKUNG 2: Die Frage „Wovon ist dieser Schmerz Ihnen bekannt?" bei Patienten, die eine Konsultation suchen, kann auch neue Informationen ergeben, die die Anamnese ergänzen._ |
| | | _ANMERKUNG 3: Die Frage „Wovon ist dieser Schmerz Ihnen bekannt?" kann beim asymptomatischen Patienten Antworten ergeben, die eindeutig nicht für die aktuelle Zeit relevant sind, oder sie kann Antworten ergeben, die eine zuvor nicht berichtete Symptomgeschichte aufdecken. Eine häufige Antwort bei asymptomatischen Personen ist, „bekannten Schmerz" mit Bezug auf einen Zahnarztbesuch oder eine zahnärztliche Behandlung zu berichten._ |
| | FALLS DIE TEMPORALIS-LOKALISATION POSITIV IST: | |
| | **Ist dieser Schmerz Ihnen bekannt von Kopfschmerzen, die Sie in diesem Bereich in den letzten 30 Tagen hatten?** | _ANMERKUNG 1: Wenn durch ein Untersuchungsverfahren Schmerzen in der Temporalis-Region erzeugt werden, wird die Frage zur Kopfschmerz-Replikation immer gestellt, unabhängig von der Antwort auf die allgemeinere Frage nach „bekanntem Schmerz". Die Antworten auf diese beiden Fragen müssen nicht identisch sein._ |
| | | _ANMERKUNG 2: Bei der Unterscheidung von „Schmerz" und „Kopfschmerz" ist die häufigste Frage des Patienten, ob es sinnvoll ist, dass sein/ihr „Kopfschmerz" bekannt ist, aber „Schmerz" nicht bekannt ist, als Reaktion auf das Untersuchungsverfahren. Es reicht üblicherweise aus, den Patienten einfach zu versichern, dass diese Antwort in Ordnung ist. Alternativ kann das relevante Untersuchungsverfahren wiederholt werden: („Möchten Sie, dass ich dieses Verfahren wiederhole?"), und dies ermöglicht dem Patienten, seine/ihre Erfahrung neu zu bewerten, wenn er/sie auf die Schmerzabfrage-Fragen antwortet._ |
| | | _ANMERKUNG 3: Wenn Patienten „Kopfschmerzen" in anderen Kaumuskelstrukturen berichten (z.B. Masseter-Region, Kiefergelenk-Region), sollte diese Abfrage zu diagnostischen Zwecken ebenfalls durchgeführt werden. Das Standard-Untersuchungsformular hat keine Antwortoption für diese Information, aber der Befund kann im Kommentarbereich des Untersuchungsformulars notiert werden._ |
| | | _ANMERKUNG 4: Ein 30-Tage-Zeitrahmen wird hier verwendet, um die Kongruenz mit dem Zeitrahmen für Schmerzen des Kausystems beizubehalten. Die International Classification of Headache Disorders, Version 2 (ICHD-2) Kriterien spezifizieren unterschiedliche Zeiträume für Seltenen, Häufigen und Chronischen Spannungskopfschmerz. Der Anwender möchte möglicherweise den Referenzzeitraum in diesem Protokoll für Kopfschmerzen ändern, um mit den ICHD-2-Kriterien konsistent zu sein. Für Kopfschmerzen sekundär zu TMD gibt es derzeit jedoch keine etablierten Zeiträume._ |
| | | _ANMERKUNG 5: Die Palpation asymptomatischer Bereiche kann eine positive Schmerzreaktion auslösen, und während der Befragung könnte ein Patient berichten, dass der Schmerz in solchen Bereichen „bekannt" ist. Es kann sein, dass der klinische Schmerz an dieser Stelle vorhanden ist und der Patient glaubte, asymptomatisch zu sein; in diesem Fall wird der Bereich als schmerzhaft unter Element 1 der Untersuchung umkodiert. Alternativ kann ein Patient berichten, dass Palpationsschmerz an einer asymptomatischen Stelle „bekannt" ist, weil diese Schmerzerfahrung ähnlich dem Schmerz ist, der in einem anderen Körperbereich erfahren wird. Die Antwort für „bekannten Schmerz" in diesem Beispiel ist |nein|, da der durch die Untersuchung provozierte Schmerz einem Schmerz bekannt sein muss, der in derselben Körperregion erfahren wurde._ |
| | Der Untersucher bestätigt, dass die in Q1 der Untersuchung angegebenen Schmerzbereiche mit den als positiv für bekannten Schmerz berichteten Körperbereichen übereinstimmen. | |
| **6.2.5** | **Abfrage übertragener Schmerz** | |
| | **Haben Sie den Schmerz nur unter meinem Finger gespürt, oder auch irgendwo anders [in anderen Bereichen Ihres Kopfes, Gesichts oder Kiefers]?** | _ANMERKUNG 1: Für die Klassifizierung von „übertragenem Schmerz" muss die Lokalisation des wahrgenommenen Schmerzes als Reaktion auf Palpationsverfahren über die Grenze der untersuchten Struktur hinausgehen. Der Untersucher bestimmt, ob die Schmerzlokalisationen eine andere anatomische Struktur oder dieselbe Struktur bezeichnen. Nur wenn der Schmerz in einer anderen anatomischen Struktur lokalisiert ist, wird er als „übertragener Schmerz" bezeichnet. Wenn Unklarheit darüber besteht, in welcher Struktur der Schmerz empfunden wird, wenn er vom Patienten als „tief" beschrieben wird, bitten Sie den Patienten, die Oberfläche des Schmerzbereichs zu lokalisieren oder, falls zutreffend, in den Mund zu zeigen._ |
| | | _ANMERKUNG 2: Diese Abfrage kann, wie in Abschnitt 5.3 angegeben, produktiv auf „nur unter meinem Finger?" verkürzt werden, für die wiederholten Durchführungen im Zusammenhang mit einer Untersuchung, die viele positive Befunde bei der Palpation aufweist. Eine alternative Kurzform ist „zieht auch woanders hin?"._ |
| | **[Zeigen Sie mir, wo Sie den Schmerz gespürt haben.] Zeigen Sie mit Ihrem Finger auf alle Bereiche, in denen Sie gerade Schmerzen gespürt haben [als Reaktion auf meinen Druck].** | |
| | Nachdem der Patient gezeigt hat, fragen Sie: | |
| | **Haben Sie den Schmerz noch irgendwo anders gespürt?** | |

## 6.3 Effiziente Durchführung

Nach den ersten positiven Reaktionen auf schmerzprovozierende Verfahren und abhängig vom Zweck der Untersuchung sowie der Fähigkeit des Patienten, leicht zwischen dem Vorhandensein von Schmerz, bekanntem Schmerz, bekanntem Kopfschmerz und übertragenem Schmerz zu unterscheiden, kann der Untersucher den Patienten anweisen, auf die Schmerzbefragung in abgekürzter Form zu antworten. Zum Beispiel könnte der Patient angewiesen werden, bei positiven Palpationsbefunden wie folgt zu antworten: „ja, bekannt" oder „ja, nicht bekannt", um die Nachfrage zu eliminieren, ob jeder positive Schmerzbefund bekannt ist oder nicht. Nachdem festgestellt wurde, dass positive Schmerzreaktionen auf mindestens 2 positive Untersuchungsverfahren dem klinischen Schmerz des Patienten bekannt sind, muss der Untersucher nur noch fragen, ob der Schmerz bekannt ist, aber nicht mehr „wovon bekannt", es sei denn, ein Bereich, der noch nicht in Untersuchungselement 1 erfasst wurde, wird als positiv auf ein Verfahren berichtet.
