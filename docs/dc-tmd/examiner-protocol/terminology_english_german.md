## Notes on Translation

1. **Formal address**: German translations use the formal "Sie" form consistently when addressing patients.

2. **Medical terminology**: Latin anatomical terms (e.g., M. temporalis, M. masseter) are retained in German medical contexts.

3. **Compound words**: German creates compound words where English uses phrases (e.g., "Schmerzlokalisation" for "pain location").

4. **Decimal notation**: German uses comma as decimal separator (e.g., "0,5 kg" instead of "0.5 kg").

5. **Consistency**: The term "bekannter Schmerz" (familiar pain) is a key diagnostic concept and should always be translated consistently.

6. **Quotation marks**: German uses „..." (Anführungszeichen) instead of "..." for quotations.
