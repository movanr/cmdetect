# Questionnaires

## Questionnaires

- DC/TMD Symptom Questionnaire (always includeed)
- Pain Drawing (Always included)
- GCPS: Graded Chronic Pain Scale
  - 1-month version
  - 3-month version
  - 6-month version (default)
- JFLS: Jaw Functional Limitation Scale (default)
- PHQ-9: Depression (optional)
- GAD-7: Anxiety (optional)
- PHQ-4: Distress (Depression & Anxiety) (default)
- PHQ-15: Physical Symptoms (optional)
- OBC: Oral Behaviors Checklist (default)
